  • 구글이 2026년 4월 20일, '제미나이 인 크롬(Gemini in Chrome)' 을 포함한 크롬 최신 AI 기능을 한국에 공식 출시
  • 최신 AI 모델 제미나이 3.1 기반으로, 데스크톱 및 iOS 환경에 먼저 제공

탭 전환 없이 웹 콘텐츠 바로 활용:

  • 크롬 사이드 패널을 통해 탭을 떠나지 않고도 현재 페이지 내용에 대해 질문·요약·분석 가능
  • 긴 웹 콘텐츠 자동 요약, 기말고사 등 시험 대비 예상 문제 생성 기능
  • "이 레시피를 비건으로 바꾸려면?" 같은 맥락 기반 질문에 즉시 답변
  • 이전에 방문한 페이지를 기억하여 나중에 다시 찾을 수 있도록 도와주어 불필요한 탭을 안심하고 닫을 수 있음
  • Mac, Windows, Chromebook Plus 사용자에게 순차적 제공

구글 서비스 통합으로 원스톱 작업:

  • 지메일 — 페이지를 벗어나지 않고 사이드 패널에서 이메일 작성 및 전송, 수정 후 클릭 한 번으로 발송
  • 구글 캘린더 — 회의 일정 즉시 등록
  • 구글 지도 — 위치 정보 바로 확인
  • 유튜브 — 시청 중인 동영상의 핵심 내용 파악 및 질문 가능
  • 브라우저 내에서 정보 확인과 후속 작업을 동시에 수행하여 업무 효율 극대화

다중 탭 컨텍스트 활용:

  • 여러 열린 탭의 정보를 동시에 분석하고 교차 확인
  • 팀 빌딩 활동 기획 시 조사한 여러 자료를 종합하여 아이스브레이킹 아이디어 제안
  • 쇼핑 시 여러 사이트의 제품 정보를 표 형태로 자동 정리하여 비교 분석 용이

나노 바나나 2 기반 이미지 변환:

  • 크롬에 나노 바나나 2 모델 내장으로 별도 파일 업로드나 새 탭 없이 즉석 이미지 변환
  • 사이드 패널에 프롬프트만 입력하면 현재 브라우저 창에서 바로 처리
  • 예: 가구 구매 전 원하는 공간에 배치해보며 인테리어 미리보기 가능
  • 복사·붙여넣기나 재업로드 불필요

보안(Security by Design):

  • 프롬프트 인젝션 등 AI 특화 위협을 식별하도록 모델을 훈련하여 웹 탐색 시 사용자 보호
  • 이메일 전송, 캘린더 일정 추가 등 민감한 작업 수행 전 반드시 사용자 확인 절차 거치도록 설계
  • 자동화된 레드팀 훈련을 통해 다층적 보안 체계를 지속 검증 및 강화
  • 자동 업데이트로 최신 위협에 신속 대응
  • 사용자가 언제든 제어권을 유지하도록 설계

향후 계획:

  • 올해 안으로 추가 기능 및 지원 지역·언어 지속 확대 예정
